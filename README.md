# Dr. Charon Blog

Source code for my personal website.

🌐 **Website:** https://blog.drcharon.com

---

## Overview

This repository contains the source code for my personal website, built with Astro and based on the Fuwari theme.

The site is deployed automatically through Cloudflare Pages.

---

## Development

Run all commands from the project root.

| Command                    | Description                          |
| :------------------------- | :----------------------------------- |
| `pnpm install`             | Install dependencies                 |
| `pnpm dev`                 | Start the local development server   |
| `pnpm build`               | Build the production site            |
| `pnpm preview`             | Preview the production build         |
| `pnpm check`               | Run project checks                   |
| `pnpm format`              | Format the project with Biome        |
| `pnpm new-post <filename>` | Create a new post                    |
| `pnpm astro ...`           | Run Astro CLI commands               |
| `pnpm astro --help`        | Display Astro CLI help               |
